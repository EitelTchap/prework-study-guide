HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The body element is generally contains several sections including a header, section, main and a footer, and so much more
  • There are semantic elements such as form, table, article
  • There are non-semantic elements such as div and span

CSS

  • display: assigns a type of display to the section
  • position relative/absolute/fixed: assigns position of section/element
  • id's start with #
  • Selector classes start with .
  • Normal tags can be written as they are
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • Typical commandlines include ls, cd, mkdir, touch, start
  • git init
  • git add: add a new file to the branch
  • git commit -m "description": commits the file to
  • git push: pushes the file to the online repository
  • git checkout -b "branch name": creates a new branch

JavaScript

  • An array is a single variable that is used to hold a group of data
  • An index is a unique number showing the location of a specific data item within an array
  • In JavaScript, arrays are zeero-indexed and sequential
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
